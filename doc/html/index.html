Violation for iOS

https://github.com/jdee/violation

Violation is a framework for iOS providing UI components including a custom gesture recognizer (ViolationDirectedPressGestureRecognizer) and a custom control that uses it (ViolationDirectionWheel). For instructions on using the framework in your build, see README.md.
